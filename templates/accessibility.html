Accessibility Commitment

{{municipality_name}} is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Conformance Status

This website is designed to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards. These guidelines explain how to make web content more accessible for people with disabilities.

Accessibility Features

  • Semantic HTML5 markup for proper structure
  • Keyboard navigation support throughout the site
  • ARIA labels and landmarks for screen readers
  • Sufficient color contrast ratios (WCAG AA compliant)
  • Responsive design that works on all devices
  • Alternative text for all images
  • Clear heading hierarchy for easy navigation
  • Focus indicators for interactive elements
  • Minimum tap target sizes (44x44 pixels)
  • Support for reduced motion preferences

Assistive Technologies

This website has been tested with the following assistive technologies:

  • Screen readers (JAWS, NVDA, VoiceOver)
  • Keyboard-only navigation
  • Browser zoom up to 200%
  • Voice control software
